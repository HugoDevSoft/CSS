Alinhamento de elementos e conteúdo usando Flexbox

Flexbox é um mecanismo do CSS3 que permite aplicar facilmente Alinhamento e distribuição de elementose e conteúdos no HTML.

Na prática, é uma técnica diferente para solucionar problemas e minimizar o grande uso de paddings, margins, floats, position na resolução de dificuldades na construção de layout.

As propriedades do flexbox são os seguintes:

  • flex-direction: Define qual direção o container deseja empilhar os itens (column, column-reverse, row. row-reverse).
  • flex-wrap: Especifica se os itens flexíveis devem ser encapsulados ou não(flex-wrap: wrap, nowrap, wrap-reverse).
  • flex-flow: Propriedades abreviada para definir flex-directiom e flex-wrap (flex-flow: row wrap).
  • justify-content: Alinhamento de itens (justify-content: center, flex-start, flex-end, space-around, space-between).
  • align-itens: Propriedades é usada para alinhar os itens flexíveis.
  • align-content: Propriedades é usada para alinhar as linhas flexíveis.
